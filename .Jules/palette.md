## 2024-05-19 - Keyboard Accessibility Focus States
**Learning:** Adding a universal `*:focus-visible` CSS rule provides a robust way to implement focus styles across a static HTML site without relying on external dependencies or altering existing markup.
**Action:** Always include keyboard focus state testing and validation in accessibility enhancements. Use `:focus-visible` in CSS instead of `:focus` to prevent unwanted focus outlines on mouse clicks, enhancing the visual experience for both keyboard and mouse users.
